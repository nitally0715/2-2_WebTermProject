# 개발 계획서

## 웹 서비스 주제

	정컴타자연습 : 프로그래머들의 전용 타자연습 및 언어 학습 웹사이트. 기능은 다음과 같다.
	1. 로그인
	2. 타자 연습 : 한컴타자연습처럼 각 언어별 타수를 측정 및 기록 / 기록 공유
	3. 함수 퀴즈 : 함수에 대한 설명을 보고 함수 이름을 맞히는 퀴즈
	4. c++, java, python 세 가지 언어중 학습하기 원하는 항목을 선택 (중복선택 시 혼합해서 나옴)

## 주제 선정 배경 및 이유

	종종 컴퓨터를 켜놓고 할 일이 없을 때 한컴타자연습을 켜서 타자연습을 하곤 한다. 그런데 일상문 대신에 코딩할 때 자주 사용하는 문장을 연습할 수 있으면 어떨까 하는 생각에서 출발하여 학생들의 코딩능력 향상 및 간단한 심심풀이의 목적으로 이번 프로젝트의 주제를 선정하게 되었다.

## 개발언어 및 사용도구/라이브러리

	- 프론트엔드 : HTML, CSS, JS
	- 백엔드 : PHP, Node.js
	- RDBMS : mySQL
	- 컨테이너 / 배포 환경 : Docker

	- 정시원 : 프론트엔드(주), 디자인
	- 조현우 : 백엔드/프론트엔드(부), 기획
