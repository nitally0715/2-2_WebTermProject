🔥 1. 타자 속도 계산 방식(CPM)
타수는 CPM(characters per minute) = (총 타자 수 / 총 시간(분))
실시간 업데이트 (100ms 또는 200ms 간격)
최종 요약에도 반영
최고 타수(best CPM)도 실시간 갱신
🔥 2. 정확도 계산
정확도 = (정타 / 전체 입력 수) × 100%
백스페이스 입력도 카운트되며 오타 후 수정 로직 반영
한컴처럼 오타 문자는 빨간색으로 표시
실시간 업데이트
🔥 3. 문장 처리 방식 변경
문장 단위 WPM 계산 삭제
문장 개수 기반 진행도 삭제
Enter 시 다음 문장 이동
최종 문장 엔터 시 요약 화면 이동
문장 리스트는 지금처럼 선택한 언어 기반으로 유지
🔥 4. UI 변경 요구사항
typing.php에서 다음 요소를 표시
▶ 현재 CPM (실시간)
▶ 최고 CPM (실시간)
▶ 정확도 (%)
▶ 입력한 문자 수
▶ 입력 시간 (초)
▶ 현재 문장
▶ 입력창
▶ 언어 라벨
